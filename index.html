Everything You Need to Clean Data

A complete toolkit for importing, cleaning, validating, and exporting your data files.

1

Smart Import

Drag & drop files or browse. Auto-detects encoding, delimiter, and data types.

  • CSV, TSV, TXT, Excel (.xlsx)
  • Auto-detect encoding (UTF-8, Windows-1252, etc.)
  • Auto-detect delimiter (comma, semicolon, tab, pipe)
  • Excel sheet selection
2

Multi-File Merge NEW

Combine multiple files into one unified dataset. Perfect for consolidating monthly reports.

  • Stack files vertically
  • Add source file column for tracking
  • Handle column mismatches (union/intersection)
  • Preview before merging
3

Powerful Cleaning

Apply transformations to standardize your data automatically.

  • Trim whitespace from all cells
  • Remove empty rows
  • Normalize 40+ null tokens (N/A, null, -, etc.)
  • Standardize headers (lowercase, underscores)
4

Auto Format NEW

Smart number and date formatting with column-level control.

  • Fix European decimals (1.234,56 to 1234.56)
  • Convert dates to ISO format (YYYY-MM-DD)
  • Per-column format settings
  • Preview before applying
5

Smart Deduplication NEW

Remove exact and fuzzy duplicates with intelligent matching algorithms.

  • Exact matching - All columns or key columns
  • Jaro-Winkler - Best for person names
  • Levenshtein - Edit distance for typos
  • Soundex/Metaphone - Phonetic matching
  • Keep strategy: first, last, or most complete
6

Data Validation

Verify data quality before exporting with configurable rules.

  • Required column check
  • Non-null validation
  • Email format validation
  • Date/numeric validation
  • Regex pattern matching
  • Unique value check
7

Diff Preview NEW

See exactly what changed before and after cleaning. Show your manager in 5 seconds.

  • Cell-by-cell comparison
  • Row addition/removal tracking
  • Color-coded output (red=removed, green=added)
  • Export to HTML or PDF
8

SQL Export

Generate SQL INSERT statements for database loading.

  • Define target table structure
  • Map columns to SQL types
  • Transaction wrapping (BEGIN/COMMIT)
  • IDENTITY_INSERT support
  • Batch GO statements

Powerful Command-Line Interface

Automate your data cleaning with scripts and batch processing.

# Clean a file using a saved profile
datacleaner clean data.csv --profile monthly --output cleaned.csv

# Remove duplicates by email column
datacleaner dedupe data.csv --columns email --keep first -o deduped.csv

# Validate emails and required fields
datacleaner validate data.csv --email email_col --required name,id --json

# Batch process all CSV files in parallel
datacleaner batch clean "data/*.csv" --profile strict --parallel 4 --output cleaned/

# Watch a folder for new files
datacleaner watch ./inbox --profile auto_clean --output ./processed

# Launch GUI mode
datacleaner --gui

clean

Apply cleaning transformations to a file.

  • --profile - Use saved profile
  • --trim - Trim whitespace
  • --normalize-nulls - Normalize null tokens
  • --dry-run - Preview changes
  • --json - JSON output for scripting

dedupe

Remove duplicate rows.

  • --columns - Key columns for matching
  • --keep - first, last, or most_complete
  • --fuzzy - Enable fuzzy matching
  • --threshold - Similarity threshold (0-1)

validate

Run validation rules on data.

  • --required - Required columns
  • --email - Email column to validate
  • --unique - Unique column check
  • --regex - Custom pattern

batch

Process multiple files at once.

  • --parallel - Number of workers
  • --continue-on-error - Don't stop on errors
  • --output - Output directory

watch

Monitor folder for new files.

  • --profile - Cleaning profile to apply
  • --output - Output directory
  • --move-originals - Archive folder

profile

Manage cleaning profiles.

  • list - List all profiles
  • show <name> - Show profile details
  • export <name> - Export to JSON
  • import <file> - Import from JSON
